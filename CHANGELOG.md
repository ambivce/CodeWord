# Change Log

## Initial release

[1.0.0] – Features Added

- Meditation Mode

  - Automatically opens selected Bible passages at configured times.
  - Supports multiple daily trigger times.
  - Randomized selection from a configurable list of books and chapters.
  - Works with both 12-hour (AM/PM) and 24-hour time formats.

- Reading Plan Mode

  - Displays scheduled Bible readings based on a structured plan.
  - Supports book-focused chapter ranges.
  - Opens readings in a dedicated editor column for focused study.

- Scheduling Engine

  - Minute-level scheduler that respects user configuration.
  - Triggers readings only when the active VS Code window is in use.
  - Prevents unnecessary background work when inactive.

- Offline Bible Support

  - Uses bundled Bible (KJV) JSON data (no internet required).
  - Fast local rendering inside VS Code webviews.

- Commands

  - CodeWord: Unsheath — manually open the Bible panel on demand.

- Configuration
  - Mode selection (Meditation or Reading Plan)
  - Multiple daily reading times
  - Meditation book list
  - Reading plan book and chapter configuration
